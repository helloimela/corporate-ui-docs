Corporate Header Code Generator

Simulate the look of your page by entering a URL of a logotype picture or if you do not have a picture, just enter a name. As long as there is a logotype picture available, the dealer name will nog be shown as text.

Code Sample

              

<c-corporate-header site-name="{{example.siteName}}" site-url="{{example.dealerUrl}}" dealer-name="{{example.dealerName}}" dealer-logo="{{example.dealerLogo}}" dealer-url="{{example.dealerUrl}}">
  <nav-item location="{{example.topLinkHref}}">{{example.topLinkName}}</nav-item>
</c-corporate-header>

Copy the code above and replace the same element it in the basic template for applications or the basic template for public sites. Depending on what template you're using, the component will get different styling applied to it.

To build your application, please use the appropriate elements from the UX Library and the Scania Bootstrap HTML Collection
