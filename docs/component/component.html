Scania Components

Use this component where applicable. If you do need for example a main navigation in your application, use the one found among these components. These are all signed of by the Scania Headquarters and there is no reason to create something similar. Do not re-invent the wheel.

Copy and Paste the code

In connection to each variation of a component there is a CODE button. Copy that code and paste it in your template page.
