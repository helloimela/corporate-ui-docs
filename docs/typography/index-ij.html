Stylesheets

<link rel="stylesheet" type="text/css" href="https://static.scania.com/resources/fonts/scania-sans/scania-sans.css">

The Scania Sans stylesheet should be linked into your application using the tag above. This ensures us that every user of the font has the latest version of it.

Cyrillic Fonts

All Scania Sans fonts are available in cyrillic versions. They are included in the Corporate UI setup. If needed for applications on a stand alone basis, then include the CSS mentioned above and the cyrillic fonts will be loaded when needed.
